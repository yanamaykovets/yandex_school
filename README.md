# yandex_school

# Portfolio for Yandex

This repository contains academic and practical works on data analysis, machine learning, numerical methods, and programming.

## Main projects
- **Customer ETL** – data cleaning, preparation, and feature engineering.
- **Text analysis** – preprocessing, lemmatization, NER, visualization.
- **Neural networks** – forward pass implementation using NumPy.
- **Gradient descent** – implementation from scratch.
- **Web scraping** – data collection from multiple websites.

Other works include lab assignments on numerical methods, databases, and algorithms.

## Tech stack
Python, pandas, numpy, matplotlib, seaborn, scikit-learn, nltk, spacy, requests, BeautifulSoup.
